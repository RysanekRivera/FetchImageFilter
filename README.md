# FetchImageFilter
<p>Downloads Images that you can apply one of several filters and then upload to server.</p>
<p>This project uses: Clean Architecture, MVVM, Retrofit, Dagger-Hilt, WorkManager, Coroutines, Flows, LiveData.</p>
